## BASIC SECTION

### 1. count_unique_elements
**Description:**  
You are given a list that may contain duplicate values. Your task is to count how many distinct elements exist.

### 2. merge_dictionaries
**Description:**  
You are given two dictionaries. Combine them into a single dictionary where values from the second dictionary override values from the first when keys overlap.

### 3. format_student_report
**Description:**  
You are given a student name and a list of marks. Produce a formatted report displaying the student’s name and their average mark.

### 4. compress_string
**Description:**  
You are given a string. Compress it by counting consecutive repeating characters and producing a compact representation.


## INTERMEDIATE SECTION

### 5. group_by_length
**Description:**  
You are given a list of words. Group them into a dictionary where each key represents a word length and the value is a list of words of that length.

### 6. reverse_list_recursively
**Description:**  
You are given a list. Reverse the list using recursion rather than loops or slicing.

### 7. rotate_matrix_90
**Description:**  
You are given a square matrix. Rotate it 90 degrees clockwise and return the resulting matrix.


## ADVANCED SECTION

### 8. flatten_nested_list
**Description:**  
You are given a nested list that may contain lists within lists. Flatten it into a single list using recursion.

### 9. sum_nested_numbers
**Description:**  
You are given a nested list of numbers. Calculate the sum of all numbers using recursion.

### 10. generate_pascal_row
**Description:**  
You are given an index n. Generate and return the nth row of Pascal’s Triangle using recursion.
